vue拖拽
2021.08.20
2021.08.21
688 words 3 min

需求

tab切换table，table行可以上下自由移动；并且tab可以左右拖动，双击tab可以编辑

概述

项目使用使用的element ui，所以在同事的推荐下，使用 vuedraggable 实现 vue tab 左右滑动
使用 el-table-draggable 实现 Vue table 的上下滑动

过程

由于要实现 tab 双击可以编辑按钮文字，单击切换 tab，所以没有使用 element ui 的 el-tabs 而是使用 el-radio-group

双击编辑 失去焦点恢复按钮模式

由于click 和double click是独立的事件，double click 的preventEvent 和stopPropagation并不能阻止click的发生,也就是说double click一定会触发click事件，所以我们监听double click事件去编辑同时也会触发tab切换事件，但是我们不想切换tab。

vue.js: how to handle click and dblclick events on same element

Handle single and double clicks on the same element in Vue.js

同时我们双击出现的input 要加上点击事件 增加 preventEvent 和stopPropagation 防止编辑的时候 tab被唤醒
另外当双击tab的时候，要让input focus，我们发现html5的autofocus属性只在第一次有效，所以我们决定加上directives指令，让input被查询的时候被唤醒

然后再el-input上加入 v-focus <el-input v-focus />
el-input失去焦点恢复btn <el-input v-focus @blur @keyup.enter.native />

最后table的数据手动控制method上下移动 置顶 只需要对data数据进行操作即可
